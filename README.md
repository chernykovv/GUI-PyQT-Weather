# PyQt5 Weather App

Modern weather app using OpenWeatherMap API, PyQt5 and CSS. 
* Determine user's location by IP address;
* Semi-transparent background;
* Checking for updates every 10 minutes.

Tools and libraries:
* Python
* PyQt5
* CSS
* requests
